# Faculdade Vencer - Sistema Acadêmico Web

Este é um **sistema acadêmico completo** desenvolvido em **HTML, CSS e JavaScript**, projetado para simular a gestão de uma faculdade moderna. Ele possui interface profissional, responsiva e funcional, adequada tanto para administração quanto para demonstração de projetos acadêmicos ou portfólio.

## 🔹 Funcionalidades

- **Home institucional** com banner, seção de cursos, notícias e sobre a faculdade.
- **Gestão de Alunos**: cadastro, atualização e listagem.
- **Gestão de Professores**: cadastro, atualização e listagem.
- **Gestão de Cursos**: cadastro de cursos com ícones, descrição e visual profissional.
- **Controle de Notas**: registro e consulta de notas dos alunos.
- **Integração com LocalStorage**: todos os dados podem ser salvos e carregados dinamicamente no navegador.
- **Layout Responsivo**: compatível com desktop, tablet e mobile.
- **Interface Profissional**: inspirado em sites institucionais de faculdades reais, com menu fixo, cards animados e ícones.

## 🔹 Tecnologias Utilizadas

- HTML5
- CSS3 (Flexbox, Grid e animações)
- JavaScript (DOM, LocalStorage)
- FontAwesome (para ícones)
- Favicons e identidade visual profissional
